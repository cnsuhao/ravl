Very preliminary support for firewire camera's using the IIDC protocol. Currently only supports the capture of grey scale images. Support for a wider range of camera's will be provided on request, or when I get my hands on some other cameras.

For a list of camera's supporting this protocol see: http://www.tele.ucl.ac.be/PEOPLE/DOUXCHAMPS/ieee1394/cameras
This driver requires the libdc1394 library which can be found at: http://sourceforge.net/projects/libdc1394
