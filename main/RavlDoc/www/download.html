What you need for Windows systems :

  • RAVL-SRC-0.7.zip
  • MSVCPP-ExternalLibs-0.7.zip

What you need for UNIX systems:

To develop applications on linux that use RAVL you'll need a minimum of:

  • RAVL-(version).rpm
  • RAVL-devel-(version).i386.rpm
  • local-gcc-2.95.3.i386.rpm

If you just want to run applications based on RAVL :

  • RAVL-(version).rpm

The files can be found on the DOWNLOAD page.

Whats available ?

The following rpm's are available for download:

RAVL-SRC-0.7.tar.bz2
RAVL-SRC-0.7.tar.gz
RAVL-SRC-0.7.zip		 Full source for RAVL, including Visual C++ project files.
RAVL-Contrib-SRC-0.7.tar.bz2
RAVL-Contrib-SRC-0.7.tar.gz
RAVL-Contrib-SRC-0.7.zip		 Source for drivers specific to particular pieces of hardware.
RAVL-(version).src.rpm The source code.
RAVL-(version).i386.rpm Binary's and shared libraries.
RAVL-devel-(version).i386.rpm Development libraries, and include files.
RAVL-doc-(version).i386.rpm Developer documentation.
RAVL-debug-(version).i386.rpm Static debug libraries, this code contains extra checks, and full debug information.
RAVL-opt-(version).i386.rpm Static optimised libraries.
MSVCPP-ExternalLibs-0.7.zip External libraries needed by RAVL. They should be unpacked into the root of the source tree.

there is also a precompiled version of gcc-2.95.3.

local-gcc-2.95.3-1.i386.rpm gcc-2.95.3 compiled into /usr/local/gcc-2.95.3, this is needed as the 2.96 compiler shipped with many linux distributions doesn't handle C++ properly.
