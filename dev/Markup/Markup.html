MarkSeq is a very simple tool to help generate manual annotation of video sequences. It is derived from VPlay, and so has the same video controls for play, single step etc. The tool is very limited in what annotation it can create; however it is straightforward to modify its behaviour to suit your particular task.

Usage:

  • To get usage, type 
    MarkSeq -help
  • To use, first you need to create a file of the set of possible annotation strings, one to a line. If you don't specify this file (option -cf), it will give you a list that was used for the Vampire tennis annotation project.
  • To generate a markup:
    1. Pause the video on the desired frame.
    2. Select one of the set of markup classes from the boxes on the right-hand side of the GUI.
    3. Click on the current frame.
    This will generate a line in the file containing the frame no., the position of the (most recent) mouse click and the index of the markup class.
  • At the top of the screen there are three numbers:
    • The middle one is the current frame.
    • The left-hand one is the frame no. of the nearest preceding annotation
    • The right-hand one is the frame no. of the nearest succeding annotation
  • The markup is save to a file specified by the (compulsory) -mf option. However, it is only saved when you press the "Save marks" button, NOT when you quit from the program. If the markup file already exists, the new markup is merged with the existing markup, so you can spread the marking up over several sessions.
